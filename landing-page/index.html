Features

📦

Single Container

One Docker container, one command install. No complex setup, no dependencies to manage.

💬

Web Interface

Chat with S3pia in your browser with a retro CRT-styled interface.

📱

Telegram

Connect via Telegram bot for mobile access and notifications.

🧠

Memory

Persistent long-term memory across conversations. Remembers context and preferences.

✨

Soul & Identity

Customizable personality that evolves over time. Make it truly yours.

⏰

Scheduled Tasks

Cron-style background jobs and reminders. Set it and forget it.

Configure in Browser

Add your API key and start chatting in seconds

Quick Start

01

Install

Run the install command. It checks Docker, clones the repo, and starts the container.

02

Configure

Open the web UI at localhost:3210, click the gear icon, and add your API key from any supported provider.

03

Chat

Start talking to your assistant. It responds instantly and handles tasks in the background.
